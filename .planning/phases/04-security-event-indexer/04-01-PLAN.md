<objective>
Create the security event indexer infrastructure and instrument all rejection points in DcapVerifier, overwatch, and consensus to emit indexed events into RocksDB.

Purpose: Persistent record of every security-relevant rejection (TEE failures, Sybil detections, CVE violations, slash events) for the block explorer to query in later phases. Valid attestations produce zero writes ("good is good, let it go").

Output: `subnet/indexer/` module with SecurityEvent dataclass, SecurityEventIndexer class, instrumented DcapVerifier/overwatch, unit + integration tests.
</objective>
